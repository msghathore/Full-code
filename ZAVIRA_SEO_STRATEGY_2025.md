### Schema Markup Implementation

Add structured data to help Google understand your business better:

#### LocalBusiness Schema (Required)
Use the **BeautySalon** schema type ([Source](https://schemantra.com/schema_list/BeautySalon)):

```json
{
  "@context": "https://schema.org",
  "@type": "BeautySalon",
  "name": "Zavira Salon & Spa",
  "image": "https://yourdomain.com/images/zavira-storefront.jpg",
  "address": {
    "@type": "PostalAddress",
    "streetAddress": "283 Tache Avenue",
    "addressLocality": "Winnipeg",
    "addressRegion": "MB",
    "postalCode": "R2H 2B2",
    "addressCountry": "CA"
  },
  "telephone": "+14318163330",
  "email": "zavirasalonandspa@gmail.com",
  "openingHoursSpecification": {
    "@type": "OpeningHoursSpecification",
    "dayOfWeek": [
      "Monday",
      "Tuesday",
      "Wednesday",
      "Thursday",
      "Friday",
      "Saturday",
      "Sunday"
    ],
    "opens": "08:00",
    "closes": "23:30"
  },
  "priceRange": "$$",
  "url": "https://yourdomain.com",
  "geo": {
    "@type": "GeoCoordinates",
    "latitude": "49.8844",
    "longitude": "-97.1308"
  },
  "sameAs": [
    "https://www.facebook.com/zavirasalon",
    "https://www.instagram.com/zavirasalon"
  ]
}
```

**Implementation:** Add this to your website header or use a plugin like [Rank Math](https://rankmath.com/) or [Yoast SEO](https://yoast.com/).

#### Service Schema
For each service page, add Service schema:

```json
{
  "@context": "https://schema.org",
  "@type": "Service",
  "serviceType": "Hair Coloring",
  "provider": {
    "@type": "BeautySalon",
    "name": "Zavira Salon & Spa"
  },
  "areaServed": "Winnipeg, MB",
  "description": "Professional hair coloring services including balayage, highlights, and full color treatments."
}
```

#### FAQ Schema
Add FAQ schema to capture featured snippets:

```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [{
    "@type": "Question",
    "name": "What are your hours?",
    "acceptedAnswer": {
      "@type": "Answer",
      "text": "Zavira Salon & Spa is open daily from 8:00 AM to 11:30 PM."
    }
  }]
}
```

> **Tools:** Use [Schema Markup Generator](https://microdatagenerator.org/localbusiness-microdata-generator/) or [Google's Structured Data Markup Helper](https://www.google.com/webmasters/markup-helper/)

### Common Pitfalls to Avoid

❌ **Don't:**
- Neglect Google Business Profile (it's the most important!)
- Copy content from competitors (Google penalizes duplicate content)
- Buy fake reviews or backlinks (severe penalties)
- Ignore negative reviews (respond professionally always)
- Use inconsistent NAP across platforms
- Forget mobile optimization (60% of traffic!)
- Publish AI content without human review
- Focus only on rankings (track conversions too)
- Give up after 2-3 months (SEO takes time!)
- Keyword stuff (write naturally for humans)

✅ **Do:**
- Be patient and consistent (3-6 month timeline)
- Focus on user experience and value
- Build genuine relationships for links
- Create original, helpful content
- Monitor and respond to all reviews
- Keep NAP perfectly consistent
- Track metrics religiously
- Adjust strategy based on data
- Stay updated on Google algorithm changes
- Combine SEO with other marketing (social, email, ads)
